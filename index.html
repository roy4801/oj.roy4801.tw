Leetcode 1870 - Minimum Speed to Arrive on Time

題目

Problem#

可以花 hour 小時通勤到辦公室，必須要按照順序搭繩 n 輛火車，並給你 dist[i] 長度 n 代表第 i 輛火車要搭幾公里
每輛列車必須在整點出發，問你能不能回傳一個最小的速度搭完所有的火車，回傳 -1 代表不可能準時到達。

測資限制#

  • $1 \le n \le 10^5$
  • $1 \le dist[i] \le 10^5$
  • $1 \le hour \le 10^9$
  • 保證答案不超過 $10^7$，且 hour 會給到小數第二位
