Leetcode 134 - Gas Station

題目

Problem#

有 n 個排成圓形的加油站，每個加油站有 gas[i] 的油，從第 i 個加油站到第 i+1 個加油站要花 cost[i] 的油
問你如果順時針走完所有的加油站，要從哪個加油站出發? (從 0 開始)，如果不能走完回傳 -1

想法#

貪心，從第 0 個加油站開始走，如果當前油量小於 0 ，就 reset 從下個加油站開始走，直到路徑總共走完 n 個加油站
