# Problem

給你兩個字串 `s1` 和 `s2` 問你最常共同子序列(LCS)長度多少?

`1 <= s1.size(), s2.size() <= 1000`

# 想法

LCS 裸題

- 時間複雜度: $\mathcal{O}(n^2)$
- 空間複雜度: $\mathcal{O}(n^2)$
