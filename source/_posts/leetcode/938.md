# Problem

給你一個二元搜尋樹，問你加總數值在 `[low, high]` 之間的 node 

# 想法

DFS 加總即可

- 時間複雜度: $\mathcal{O}(n)$
- 空間複雜度: $\mathcal{O}(1)$
