# Problem

![](https://assets.leetcode.com/uploads/2020/08/21/largest_e1.jpg)

給你一個二元樹，要你輸出每一層的最大值。

## 測資限制

- 節點數量：$0 \le n \le 10^4$
- $-2^{31} \le val \le 2^{31}-1$

# 想法

dfs 帶層數把每層存出來 sort 即可
