# Problem
題目給你排序好的陣列 `nums` 和整數 `target`，問你如果 `target` 存在於 `nums` 中則回傳 index；反之，回傳他可能插入的第一個 index

# 想法

其實是問你陣列中第一個 >= target 的數字之位置，二分搜裸題

- 時間複雜度: $\mathcal{O}(\log{n})$
- 空間複雜度: $\mathcal{O}(1)$
