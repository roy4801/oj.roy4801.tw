# Problem

給你一個雙向圖有 $n$ 個點 $m$ 條邊，從 $0$ 到 $n-1$，問你能不能從 `src` 走到 `dst` 

- $1 \le n \le 2e5$
- $0 \le m \le 2e5$
- 沒有重複的邊 or 自環

# 想法

disjoint set $\approx \mathcal{O}(1)$

- 時間複雜度: $\mathcal{O}(m)$ m = number of edges
- 空間複雜度: $\mathcal{O}(n)$
