# Problem

題目給你一個陣列 `nums`，回傳出現次數超過 $\lfloor \frac{n}{2} \rfloor$ 的的那個數字。

# 想法

直覺做就是先建表紀錄每個數字出現的次數，接著排序拿最大即可

- 時間複雜度: $\mathcal{O}(N\log{N})$
- 空間複雜度: $\mathcal{O}(N)$

```cpp linenum="1"
class Solution {
public:
    int majorityElement(vector<int>& nums)
    {
        int vote = nums[0], cnt = 1;
        
        for(int i = 1; i < nums.size(); i++) {
            if(nums[i] == vote) cnt++;
            else cnt--;
            
            if(cnt == 0) {
                vote = nums[i+1];
                cnt = 1;
                i++;
            }
        }
        
        return vote;
    }
};
```
