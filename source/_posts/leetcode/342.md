# Problem

給你整數 `n` 如果 $n = 4^x$ 的話回傳 `true` 反之回傳 `false`

Follow up: $O(1)$ 時間複雜度

## 測資限制

- $-2^31 \le n \le 2^31-1$

# 想法

判斷一個整數是不是某個數(`x`)的次方可以除除看，或是取 $\log{x}$
