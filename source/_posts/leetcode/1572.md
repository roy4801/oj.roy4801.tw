# Problem

![](https://assets.leetcode.com/uploads/2020/08/14/sample_1911.png)

給你一個矩陣 `mat` 要你回傳對角的總和，重複的只要加一次就好

- 測資限制:
    - $1 \le n \le 100$
    - $mat = nxn$
    - $1 \le mat[i][j] \le 100$

# 想法

- 時間複雜度: $\mathcal{O}(n)$
