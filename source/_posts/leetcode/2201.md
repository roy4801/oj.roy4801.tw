# Problem

題目給你一個 n x n 的 grid 並給你一堆寶藏的地點 `artifact[i] = [r1_i, c1_i, r2_i, c2_i]` 其中 `(r1, c1)` 與 `(r2, c2)` 代表寶藏的左上到右下
接著給你挖掘的座標 `dig[i] = [r_i, c_i]` 問你最後挖了幾個寶藏(一個寶藏要被挖出 <=> 挖出寶藏所佔的所有格子)

# 想法

- 時間複雜度: $\mathcal{O}(n^2)$
- 空間複雜度: $\mathcal{O}(n^2)$
