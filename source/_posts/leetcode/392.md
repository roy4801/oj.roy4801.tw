# Problem

給兩個字串 `s` 和 `t` 問 `s` 是不是 `t` 的子序列(Subsequence)

- `0 <= s.length <= 100`
- `0 <= t.length <= 104`

Follow-up: 當 s.length >= 10^9 時要怎麼做

# 想法

當下想到 DP 的作法，看 `s` 和 `t` 的 LCS 長度是否跟 `s.length()` 一樣就好，$O(NM)$
另一個做法就是雙指標直接看 `s` 是否都在 `t` 裏頭就好 $O(N+M)$

`N=s.length`, `M=t.length`
