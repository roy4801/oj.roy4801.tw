# Problem

給你一個[逆波蘭表達式(Reverse Polish notation)](https://zh.wikipedia.org/wiki/%E9%80%86%E6%B3%A2%E5%85%B0%E8%A1%A8%E7%A4%BA%E6%B3%95)

# 想法

用一個 stack 即可，遇到數字 push 到 stack 中，遇到 op 則 pop 兩個數字然後計算後再 push 回去

- 時間複雜度: $\mathcal{O}(n)$
- 空間複雜度: $\mathcal{O}(n)$
